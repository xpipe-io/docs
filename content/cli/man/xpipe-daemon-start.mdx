## Synopsis

**xpipe daemon start** [**--port**=_&lt;port>_] [_&lt;input>_...]


## Description

Starts the XPipe daemon if it is not running


## Options

**--port**=_&lt;port>_::
  The HTTP server port


## Arguments

[_&lt;input>_...]::
  Optional arguments to pass to the daemon
