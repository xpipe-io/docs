## Synopsis

**xpipe daemon mode** [**--port**=_&lt;port>_] _&lt;mode>_


## Description

Switches the operation mode of the XPipe daemon


## Options

**--port**=_&lt;port>_::
  The HTTP server port


## Arguments

_&lt;mode>_::
  The mode to switch to. The mode id must be [background, tray, gui]. Note that not necessarily all operation modes are supported on your current platform.
