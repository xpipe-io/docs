title: xpipe-daemon(1)
description: Commands for controlling the XPipe daemon
---

## Synopsis

**xpipe daemon** &lt;subcommand>

## Commands

[**start**](xpipe-daemon-start.adoc)::
  Starts the XPipe daemon if it is not running

[**mode**](xpipe-daemon-mode.adoc)::
  Switches the operation mode of the XPipe daemon

[**status**](xpipe-daemon-status.adoc)::
  Reports the current status of the XPipe daemon

[**stop**](xpipe-daemon-stop.adoc)::
  Stops the XPipe daemon if it is running.
